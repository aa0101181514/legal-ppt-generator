# 法庭風格 PPT 大綱範本

本文件定義律師於法庭言詞辯論、論告、辯論終結、聲請會議等場合所用之「簡報大綱」標準結構。本範本以刑事辯護為主、行政訴訟與民事為輔，可依案件調整。

---

## 通則

### 章節編號

依下列順序，**先程序後實體**：

```
封面
壹、犯（作）後態度 / 前置事由
貳、程序爭點（傳聞證據、證據能力、管轄、程式瑕疵）
參、第一層實體爭點（與本案定罪/勝敗最相關）
肆、第二層實體爭點
伍、第三層實體爭點
…
結論與聲明（求為判決 / 求為駁回 / 求為停止執行等）
```

### 每張投影片組成

```markdown
## 【Slide N】章節編號 — 標題

**核心主張**：一句話結論

**論述**：
- 論點 1
- 論點 2
- 論點 3

**支持依據**：
- 判決：[最高法院 108 年度台上字第 2027 號判決](https://api.dr-lawbot.com/api/search?q=108+台上+2027)（已驗證 ✓）
  > 逐字引用 main_preview 或 snippet 片段

**卷證引用**（若有）：
- 115 金重訴 1 號卷第 X 頁（即起訴書第 Y 頁）
```

## 刑事辯護簡報結構範本

### 封面

- 案號（例：高雄地院 115 年度金重訴字第 1 號）
- 辯護對象（例：被告 OOO、OOO）
- 被訴罪名
- 報告律師 / 事務所
- 報告日期

### 壹、犯後態度

- 補繳稅額（若屬稅捐案件）
- 配合調查情形（提供帳冊、出席偵訊）
- 認罪協商或自白內容
- 被告年齡、身分、家庭背景（量刑因素）

### 貳、程序爭點

以下為常見程序爭點，依案件挑選：

#### 2.1 傳聞證據排除（§159）
- 核心：哪些筆錄 / 調查報告 / 函文屬傳聞，應予排除
- 支持判決：尋找「刑事訴訟法 第 159 條 傳聞法則」相關最高法院判決

#### 2.2 毒樹果實原則
- 核心：前階段違法取證是否污染後續證據
- 支持判決：尋找相關最高法院見解

#### 2.3 舉證責任
- 核心：檢察官負實質舉證責任
- 支持判決：最高行政法院 107 判字 369 號（行政訴訟）、最高法院相關判決

### 參~陸、實體爭點（依案件類型）

#### 稅捐案件（以 ABC 牙醫案為例）

- 重複課稅：薪資所得 vs 執業所得雙重計入之違誤
- 所得性質爭議：薪資 vs 執業所得 vs 股利
- 合夥事實認定：勞務出資、共負盈虧、自由排班、親自履行
- 核課期間：5 年 vs 7 年，舉證門檻「幾近於真實之蓋然性」
- 不確定故意 vs 詐術要件（§41 稅捐稽徵法）

#### 詐欺 / 背信類

- 財產上交付 vs 單純債務不履行之區別
- 主觀構成要件「意圖為自己或第三人不法所有」
- 行為人與結果間因果關係
- 共同正犯或幫助犯

#### 組織犯罪條例

- 三人以上有結構性組織之認定
- 持續性、牟利性
- 參與行為程度

### 結論

- 聲明事項
  - 請求判決：例「被告 OOO 無罪」、「原判決撤銷」、「原處分停止執行」
  - 證據調查：尚未調查之證據請求
  - 量刑意見（若退一步言）

### 貳、程序爭點
- 核課期間是否逾 5 年

- 所得性質
- 舉證責任分配

- 推計課稅之合理性

### 肆、結論
- 求為判決：原處分及訴願決定均撤銷

## 投影片字數與視覺原則

- 每張投影片標題**不超過 20 字**
- 主論述 bullet**不超過 3 層**
- 判決引文 blockquote**不超過 100 字**
- Markdown 中用 `---` 分隔投影片
- 重點字詞可用 `**粗體**`，避免使用底線
- 絕對不要用 emoji 在投影片本文（可用於 checklist 型內部筆記）

---

## 產出格式

最終交付給使用者的 Markdown 檔應包含：

1. YAML frontmatter（可選，若使用 Gamma 自動偵測）
2. 封面 slide
3. 目錄 slide（Table of Contents，若超過 10 張）
4. 內容 slides（依上述結構）
5. 結論 slide
6. 附錄：判決清單（所有引用判決字號 + 驗證連結彙整）

---

## 給 Gamma.app / Google Slides 使用者的提示

- Gamma.app 直接把整份 Markdown 貼進「Paste in text」即可。
- Google Slides 可用 Markdown-to-Slides 類擴充套件，或手動複製每張投影片內容。
- Canva 需要先轉成 PPT 結構，使用者可先匯出 Gamma 的 pptx 再匯入 Canva。
